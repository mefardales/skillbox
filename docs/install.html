Installation Guide

Get up and running in seconds. No configuration required — Skillbox detects your AI tools and installs skills to the right location automatically.

Quick Start

Start in seconds

No global install required. Run skills with npx, or install globally with npm.

CLI — npx (no install needed)
Install a skill immediately 
# No install needed — runs directly with npx
npx skillbox install frontend/react-components
Or install globally 
# Install globally so you can run `skillbox` anywhere
npm install -g skillbox

# Then use it without npx
skillbox install frontend/react-components
Auto-detected install paths
Claude Code→~/.claude/skills/
Cursor→~/.cursor/skills/
Codex→~/.codex/skills/
Project→./.<tool>/skills/
VSCode / Cursor Extension
1
Open the Extensions panel
Press Ctrl+Shift+X (or Cmd+Shift+X on Mac) in VSCode or Cursor.
2
Search for "Skillbox"
Find the official Skillbox extension and click Install.
3
Open the Skillbox panel
Click the Skillbox icon in the Activity Bar to browse and install skills visually.
Or install via CLI 
code --install-extension skillbox.skillbox

CLI Reference

All commands

Complete reference for every Skillbox CLI command.

skillbox install <skill>
Install a skill to all detected AI tools. Use --project to install locally, or --tool <name> to target a specific tool.
$ npx skillbox install frontend/react-components
$ npx skillbox install backend/django --project
$ npx skillbox install devops/docker --tool cursor
skillbox list
List all available skills in the registry grouped by category. Use --installed to show only installed skills.
$ npx skillbox list
$ npx skillbox list --installed
$ npx skillbox list --category backend
skillbox search <query>
Search skills by name, description, or tags. Returns ranked results matching your query.
$ npx skillbox search react
$ npx skillbox search "database design"
$ npx skillbox search kubernetes
skillbox detect
Detect which AI coding tools are installed on your system and show their install paths for skills.
$ npx skillbox detect
skillbox remove <skill>
Remove an installed skill from all detected AI tools. Use --tool to remove from a specific tool only.
$ npx skillbox remove frontend/react-components
$ npx skillbox remove backend/django --tool cursor
skillbox update
Update all installed skills to the latest version from the registry.
$ npx skillbox update
$ npx skillbox update frontend/react-components
Full install workflow example 
# 1. See which tools Skillbox can install to
npx skillbox detect

# 2. Find skills for your stack
npx skillbox search react

# 3. Install the skills you need
npx skillbox install frontend/react-components
npx skillbox install frontend/nextjs-app-router

# 4. Confirm they're installed
npx skillbox list --installed
Configuration

Environment variables

Configure Skillbox behavior with environment variables. All are optional — Skillbox works with sensible defaults.

Set these in your shell profile (~/.bashrc, ~/.zshrc) or pass them inline when running commands.
Variable Description Default
SKILLBOX_REGISTRY URL or local path to a custom skill registry. Use for local development or private registries. https://registry.skillbox.dev
SKILLBOX_TOOLS_DIR Override the base directory where tools are detected. Useful in CI/CD or non-standard setups. ~/ (home directory)
SKILLBOX_NO_COLOR Disable colored terminal output. Useful for log aggregation or CI environments. false
SKILLBOX_DEBUG Enable verbose debug logging for troubleshooting install issues. false
Local development with a custom registry 
# Point Skillbox at a local skills directory
SKILLBOX_REGISTRY=./local-skills npx skillbox install backend/my-custom-skill

# Or export it for the session
export SKILLBOX_REGISTRY=./local-skills
npx skillbox list
Requirements

System requirements

Node.js 18+
Skillbox requires Node.js 18 or later. Install from nodejs.org or use nvm.
macOS / Linux / WSL
Works on macOS and Linux. Windows is supported via WSL 2. Native Windows support is planned.
At least one AI tool
Skills install into Claude Code, Cursor, Codex, or any tool supporting the Agent Skills standard.
Skillbox

A curated registry of AI coding skills. Install expert instructions with one command.
